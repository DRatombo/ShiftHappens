## 🚀 **HOW TO RUN**

### **Option 1: Quick Start (Windows)**
```bash
# Double-click deploy.bat
# OR run in terminal:
deploy.bat
```

### **Option 2: Manual Start**
```bash
# Install dependencies
npm install

# Start server
node server.js

# Open browser to:
http://localhost:3000
```

### **Option 3: Development Mode**
```bash
# Install dependencies
npm install

# Start with auto-restart
npm run dev
```
